# Sui Workshop Projects

Berikut adalah gambaran umum (garis besar) dari setiap project yang ada di dalam workspace ini:

## Contract (Backend)

*   **`sui_nft`**
    *   Membuat dan mengatur NFT (Non-Fungible Tokens).
    *   Contoh fungsi: Minting NFT dasar.

*   **`sui_token`**
    *   Membuat dan mengatur Coin/Token di jaringan Sui.
    *   Contoh fungsi: Minting coin, pengaturan treasury cap.

*   **`battle_nft`**
    *   Implementasi NFT yang lebih kompleks, kemungkinan untuk mekanisme game atau pertarungan.
    *   Contoh fungsi: NFT dengan atribut yang bisa berubah atau berinteraksi.

*   **`sui_helloworld`**
    *   Project dasar untuk pengenalan Move di Sui.
    *   Fungsi sederhana untuk mengetes deploy dan call.

## Frontend

*   **`fe-day2`**
    *   Aplikasi website untuk berinteraksi dengan smart contract (module) di atas.
    *   Fitur: Connect wallet, memanggil fungsi mint, transfer, dll.

## Tools

*   **`sui-mainnet-v1.64.2-windows-x86_64`**
    *   Berisi binary executable untuk menjalankan CLI Sui.
